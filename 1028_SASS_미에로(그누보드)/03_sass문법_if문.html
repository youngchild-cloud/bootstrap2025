if 문법
- 조건에 따른 결과를 true/false로 나눠서 처리하는 삼항연산자와 비슷한 문법입니다.
- 조건식이 true이면 '참', 그렇지 않으면 '거짓'으로 반환합니다.
    문법)
    if(조건식, 참인 값, 거짓인 값);

    //삼항조건 연산자
    조건식?참표현식:거짓표현식;

    }@else{
      //위 조건 모두 거짓일 경우 표현식
    }
